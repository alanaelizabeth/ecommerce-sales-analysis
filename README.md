# E-commerce Sales Analysis

## Objective
Analyze e-commerce sales data to identify trends, evaluate business performance, and generate actionable insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Superstore sales dataset containing information about sales, profit, categories, and regions.

## Analysis Performed
- Sales analysis by category
- Region-wise performance analysis
- Profit analysis across categories
- Identification of loss-making sub-categories

## Key Insights
- Technology category generates the highest sales
- West region performs the best overall
- Some sub-categories are generating losses
- High sales do not always correspond to high profit

## Recommendations
- Focus on profitable categories
- Improve performance in low-performing regions
- Reduce losses in weak sub-categories
- Optimize pricing and cost strategies
